## Installation

```bash
# Install
$ npm install prism-nsis

# Build
$ cd prism-nsis
$ yarn || npm install
```

## Usage

### Build

A minified JavaScript package will be built automatically after installing with NPM. To manually trigger a build, run `npm run build`.
